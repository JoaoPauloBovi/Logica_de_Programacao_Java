Lógica de Programação com Java

Este repositório contém a resolução de exercícios e algoritmos desenvolvidos durante o curso de Lógica de Programação, utilizando a linguagem Java como base. O foco principal é o desenvolvimento do raciocínio lógico e a correta estruturação de soluções computacionais.
Conteúdo Programático

    Fundamentos: Variáveis, tipos de dados e operadores.

    Estrutura Condicional: Controle de fluxo (if-else, switch-case).

    Estruturas Repetitivas: Implementação de laços (while, for, do-while).

    Processamento de Dados: Entrada e saída de dados, manipulação de strings e funções matemáticas.

Objetivo

Consolidar a base lógica necessária para a transição para a Programação Orientada a Objetos e o desenvolvimento de sistemas complexos no contexto de ADS.
